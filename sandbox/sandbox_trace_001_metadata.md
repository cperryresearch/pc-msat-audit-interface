# Sandbox Trace 001

## Source

Barn Swallow GPS trajectory fragment used within the Structured Orb Dynamics
(SOD) ecosystem for demonstration and testing purposes.

The fragment is derived from the Barn Swallow dataset previously used in
PC-MSAT / SOD examples.

## Trace Characteristics

- Trace length: ~20 points
- Sampling cadence: inherited from source dataset
- Motion character: straight-dominant
- Intended regime behavior: contiguous persistence

## Purpose

This trace serves as the **baseline sandbox example**.

It demonstrates a simple, clearly persistent motion structure and is
expected to produce a **PROCEED** audit outcome when processed by the
canonical PC-MSAT renderer.

The example provides a minimal real-world fragment used to verify that
the audit pipeline behaves as expected under straightforward conditions.

## Notes

This artifact is part of the sandbox workspace and is **non-canonical**.
It exists solely for exploratory and pedagogical purposes.

The canonical audit mechanism remains defined by the renderer and
formal specification contained elsewhere in the repository.
